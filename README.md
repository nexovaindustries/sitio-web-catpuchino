# CATPUCHINO - Caricias y Cafés

Bienvenido al repositorio oficial del sitio web de **CATPUCHINO**.

Este proyecto es una página web moderna, vibrante y "kawaii" para un café de gatos, diseñada para ofrecer una experiencia visual encantadora y funcional.

## Características
- **Diseño Kawaii**: Estética inspirada en Harajuku con colores pastel y animaciones suaves.
- **Responsivo**: Optimizado para dispositivos móviles y escritorio.
- **Secciones**:
  - Hero con animaciones de gatos flotantes.
  - Historia del café.
  - Vista previa del menú.
  - Formulario de reservas interactivo.
- **Tecnologías**: HTML5, Tailwind CSS, Google Fonts, Material Symbols.
